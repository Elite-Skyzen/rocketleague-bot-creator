Rocket League Bot Creator

Instructions après téléchargement :

1. Ouvre l’application Bot Runner.

2. Clique sur “Choisir le bot” et sélectionne le fichier ZIP que tu viens de télécharger.

3. Les caractéristiques du bot (niveau, voiture, compétences...) seront automatiquement appliquées.

4. Clique sur “Lancer le match” pour démarrer le jeu avec le bot.
