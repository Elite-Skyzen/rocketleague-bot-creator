Rocket League Bot Creator

Comment utiliser le bot téléchargé

  1. Décompresse le fichier rocketleague_bot_ready.zip (si ce n’est pas encore fait).
  2. Place le fichier settings.json que tu viens de télécharger dans ce dossier.
  3. Ouvre un terminal dans ce dossier.
  4. Lance la commande : python3 launch_bot.py
  5. Rocket League va se lancer avec ton bot personnalisé !
